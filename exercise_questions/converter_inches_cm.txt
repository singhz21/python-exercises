Write a program that will convert a user entered value from inches to centimeters.

The program will:
 - ask the user to enter a value in inches.
 - wait until the user enters a value and presses the "Enter" key.
 - take the value entered by the user and calculate/convert it to centimeters.
 	Hint: 1 inch is 2.54 centimeters
 - display the result of conversion in the following format:
	[user entered value] inches is [result of conversion] centimeters
